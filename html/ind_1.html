Інструкція по застосуванню інтерактивного веб-сервісу фільтрації зображень

Веб-сервіс надає можливість працювати в двох режимах. Перший режим дозволяє проводити дослідження якості фільтрації різними методами (медіанна фільтрація, фільтрація авторським методом найменших кінцевих різниць (МНКР) або модифікованим методом - ММНКР) зображень, в які користувач може самостійно додати шум різного типу (імпульсні шуми, шуми градацій сірого) та різної інтенсивності (заповнення шумом від 0% до 100%). Другий режим дозволяє проводити фільтрацію зображень вже зашумлених фотографій, які можна завантажити із зовнішніх джерел при застосуванні кнопки "Обрати файл", яка знаходиться зверху головної сторінки.

Інтерфейс розробленого веб-сервісу надає можливість вибору зображень зі списку або завантаження довільного графічного файлу для здійснення процедур фільтрації. Для цього необхідно:

1. Обрати файл для фільтрації: - завантажити з репозиторію, в якому вже є відповідні файли, або скористатитися завантаженням з локального диску (або локального сховища, наприкалд для смартфонів).

2. Перейти в поле "Зашумлення зображення". Для роботи в першому режимі користувач може самостійно додати адитивний шум наступних типів: «Сіль» - випадкові викиди максимальної яскравості - білі точки (обрати відсоток заповнення шумом - від 0 до 100%); «Перець» - випадкові викиди мінімальної яскравості - чорні точки (обрати відсоток заповнення шумом - від 0 до 100%); «Градації сірого» - випадкові викиди градацій сірого (обрати відсоток заповнення шумом - від 0 до 100%). Після встановлення значень натиснути відповідну кнопку.

3. Передбачена можливість створити шум для кольорового зображення (опція "Шум"), стиль зображення типу «Сепія». Після таких маніпуляцій утворюється зашумлене ч/б зображення, яке підлягає фільтрації.

4. В залежності від створеного шуму або його наявності на зображенні (у випадку завантаження зашумленого зображення), перейти відповідно до поля "Фільтрація зображення з шумом типу "Сіль", "Фільтрація зображення з шумом типу "Перець" або "Фільтрація зображення з шумом типу "Градації сірого".

5. Завантажити зображення у відповідне поле (натиснути кнопку "Завантажити") або стерти його (кнопка "Очистити").

6. Для реалізації класичної медіанної фільтрації натиснути кнопки: (Медіанний фільтр "4") (реалізується фільтр типу «Медіана-4») або (Медіанний фільтр "8") (реалізується фільтр типу «Медіана-8») для можливості порівняння з результатами фільтрації при застосуванні відомих програмних додатків (наприклад, Adobe PhotoShop, GIPM).

7. Для реалізації авторської фільтрації методом найменших кінцевих різниць (МНКР) встановити в полі введення даних значення від 1 до 255 (кількість ітерацій над одним пікселем зображення) і натиснути кнопки: (МНКР 1) (реалізується однократний алгоритм методу МНКР) або (МНКР 3) (реалізується тричі алгоритм методу МНКР). Окрім того, користувач може встановлювати такі параметри фільтрації для МНКР, як кількість сусідніх пікселів, які підлягають обробці (ядро фільтраціції). За замовчуванням приймається число "8" (рекомендується), але є можливість встановити на "4" (для деяких типів зашумлених зображень дає кращі результати, ніж для "8" ).

8. Результат фільтрації користувач може спостерігати у відповідному вікні на головному полі після появи заповненого «Progress Bar», а також якість фільтрації по критерію середнього квадратичного відхилення «СКВ» в полі «Ефективність фільтрації».

9. Для реалізації фільтрації адитивного Гаусового шуму (приклад такого зображення можна завантажити готовою опцією на головному меню - сама верхня опція розташованого списку на головній сторінці) застосовується модифікований метод МНКР - викликається при натисканні кнопки ММНКР. Для роботи алгоритму за цим вдосконаленим методом користувач має встановити такі параметри:
- значення амлітуди в межах -255 до +255;
- кількість сусідніх пікселів фільтрації: "4" або "8". За замовчування обирається "8";
- глибина фільтрації - кількість ітерацій, протягом яких відбуваються зміни значень пікселів зображення. Встановлюється множник n від 1 до 15, який означає n*10^6 разів);
- знак фільтрації, який може приймати значення "+", "-", "+/-" - градієнт зміни випадкового значення пікселя при фільтрації. По замовчуванню встановлюється в "-". Наприклад, при фільтрації шуму типу "Гранулярний білий шум" рекомендується понизити яскравість білих плям, тому градіентом буде значення "-", якщо на зображенні присутні чорні плями - обирають "+", якщо присутні білі і чорні плями - обирають "+/-".
